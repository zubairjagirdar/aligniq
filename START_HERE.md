## 🎓 Learn by Doing

### Modify a Feature
- Understand: Check `src/types/index.ts` for data model
- Extend: Add method in relevant service (`src/services/`)
- Wire: Add API endpoint in `src/app/api/`
- Display: Create component in `src/components/`
- Test: Check component displays correctly

### Add a New API Endpoint
1. Define types in `src/types/api.ts`
2. Create handler in `src/app/api/`
3. Call service layer
4. Return typed response
5. Document in `ARCHITECTURE.md`

### Customize Styling
1. Edit colors in `tailwind.config.cjs`
2. Modify glassmorphism in `src/styles/globals.css`
3. Update component classes in `src/components/`
4. Test in all themes (light/dark)

### Change Database Schema
1. Edit `prisma/schema.prisma`
2. Run `pnpm prisma migrate dev --name feature_name`
3. Update types in `src/types/index.ts`
4. Update services that use model
5. Test locally before deploying
